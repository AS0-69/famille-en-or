# La Famille en Or - Jeu Web Interactif

Développeur : Ahmet-Selim Ozsoy

Version : 2.1

Bienvenue dans l'adaptation web du célèbre jeu télévisé "Une Famille en Or". Ce projet est un jeu interactif conçu pour animer des soirées et des événements communautaires directement depuis un navigateur web.

## 🕌 Contexte et Remerciements

Ce projet a été spécialement développé pour la **CIMG Mosquée Bleue de Villefranche-sur-Saône** et son organisation de jeunesse (**Gençlik Teşkilatı**).

Il a pour but de divertir et de rassembler la communauté autour d'une activité ludique et conviviale.

## 📂 Structure du Projet

Le projet respecte une architecture propre et organisée, séparant la logique (JS), le style (CSS) et la structure (HTML).
